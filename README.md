# Installation

## Android
1. yarn
2. npx react-native link
3. npx react-native run-android

## iOS
1. yarn
2. npx react-native link
3. npx pod-install
4. npx react-native run-ios

# Linting
`npm run lint` for report.  
`npm run lint -- --fix` to fix issues.
