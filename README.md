# Installation

## Android

1. yarn
2. npx react-native link
3. npm run android

## iOS

1. yarn
2. npx react-native link
3. npx pod-install
4. npm run ios

# Linting

`npm run lint` for report.  
`npm run lint -- --fix` to fix issues.
